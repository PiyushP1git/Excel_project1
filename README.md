# Vrinda Store Sales Analysis (Excel Dashboard)

This project analyzes retail sales data using Microsoft Excel to generate business insights.

## Dashboard Features
- Orders vs Sales trend analysis
- Sales distribution across states
- Channel-wise sales breakdown
- Customer demographics (Age & Gender)
- Order status analysis

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Graphs
- Slicers & Filters

## Key Insights
- Identified top-performing states contributing to revenue
- Analyzed customer purchasing behavior based on gender and age
- Evaluated performance of different sales channels
- Observed monthly sales trends and order patterns
